  فدك اللوجستية — نظام إدارة الأسطول
  Fadak Logistics Fleet Management System
═══════════════════════════════════════════════════

كيفية التشغيل:
1. تأكد من تثبيت Python 3
2. افتح Terminal/Command Prompt في مجلد البرنامج
3. شغّل: python app.py
4. افتح المتصفح على: http://localhost:5000

الملفات:
- app.py      : الخادم والقاعدة البيانات
- index.html  : واجهة المستخدم
- fadak.db    : قاعدة البيانات (تنشأ تلقائياً)

البيانات محفوظة في: fadak.db (SQLite)
